# Legal Case Repository Web App — QA Portfolio Sample

## Overview

This portfolio sample demonstrates QA testing work performed on a web-based **Legal Case Repository** application. The application is designed to help users browse, search, filter, view, and export legal case records.

This package is intended for freelance job applications, especially for roles involving:

- Manual QA Testing
- Web Application Testing
- Functional Testing
- Regression Testing
- API / Data Validation Testing
- Test Case Design
- Bug Reporting
- Release Validation
- Azure DevOps / QA Process Support

All screenshots and documents are watermarked as portfolio samples to discourage unauthorized copying.

## Project Used for Testing

**Project Name:** Legal Case Repository Web App  
**Application Type:** Web-based legal case database  
**Sample Use Case:** Browsing and managing legal case digests and related case information  
**Testing Focus:** UI behavior, search/filter functionality, case details, export feature, data handling, and admin visibility
